# Recipe Repository 
Introducing Recipe Repository – your personal digital cookbook! Designed for food lovers of all levels, this app allows you to store, organize, and access all your favorite recipes in one convenient place. This app allows you to sort through recipes by category (ex. Breakfast, Lunch, Dinner, etc) for greater convenience, and allows you to add links to existing videos and recipes from other websites. 

## How It's Made
### Tech Used: Python, Javascript, HTML, CSS, React, Flask
**Why was this stack chosen?** 
- **Full-Stack Development**: Using this stack allows you to develop both the front-end and back-end of your application, ensuring seamless integration and communication between the two.
- **Efficient Development**: The combination of Python/Flask for the back-end and React for the front-end provides a robust, efficient, and scalable way to handle CRUD operations.
- **Community and Resources**: Each of these technologies has a large and active community, meaning you have access to extensive resources, tutorials, and support.
- **RESTful API**: Flask can easily be used to create RESTful APIs, which React can consume to perform CRUD operations, providing a clear separation of concerns and a scalable architecture.
